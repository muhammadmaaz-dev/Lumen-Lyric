## ‚ú® Key Features

* **üì• In-Built Music Downloader**
  Paste a YouTube link and convert it into MP3 audio.

* **üéß Local Media Player**
  Play music directly from your device storage.

* **üîì Royalty-Free Streaming**
  Discover and stream copyright-free music.

* **üîê User Authentication**
  Secure login using Google or Email.

* **üåó Dark / Light Mode**
  Fully supported theme switching.

* **üìÇ Library Management**
  Manage liked songs, downloads, and playlists.

## üõ†Ô∏è Tech Stack

This project is built using **Flutter** and the following technologies are currently used in the codebase.

### Core
- **Framework:** Flutter (Dart)
- **State Management:** Riverpod (`flutter_riverpod`)
- **Annotations / Code Generation:**  
  `riverpod_annotation`, `freezed_annotation`

### Audio & Media
- **Audio Playback:** `just_audio`
- **Local Audio Query:** `on_audio_query`
- **Audio Metadata:** `flutter_audio_tagger`
- **Playback Progress UI:** `audio_video_progress_bar`
- **Mini Player UI:** `miniplayer`

### UI / UX
- **Icons:** `cupertino_icons`
- **Responsive Layout:** `flutter_screenutil`
- **Animations:** `lottie`, `flutter_animate`
- **Scrolling Text:** `marquee`
- **Custom Sliders:** `interactive_slider`

### Storage & Permissions
- **Local Storage:** `shared_preferences`
- **Permissions:** `permission_handler`

### Transitional
- **Provider:** `provider`  
  _(Used in some parts of the app; gradually being migrated to Riverpod)_

> ‚ö†Ô∏è Tech choices are **not final** and may change as the project evolves.  
> Contributors are welcome to suggest better alternatives via Issues.

## üöÄ Getting Started

> ‚ö†Ô∏è This project is **Under Construction** and actively welcoming contributors.

### Prerequisites

* Flutter SDK
* Android Studio or VS Code

### Installation

1. **Fork** the repository.

2. **Clone** the project:

   ```bash
   git clone https://github.com/muhammadmaaz-dev/Lumen-Lyric.git
   ```

3. **Navigate to the project directory:**

   ```bash
   cd musicapp
   ```

4. **Install dependencies:**

   ```bash
   flutter pub get
   ```

5. **Run the app:**

   ```bash
   flutter run
   ```
